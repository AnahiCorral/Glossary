Glosario de Términos HTML y Relacionados

A

API
Interfaz de programación para conectar diferentes sistemas.
Accesibilidad
Práctica de hacer los sitios web utilizables para todas las personas, incluidas aquellas con discapacidades.
Atributos
Propiedades adicionales de las etiquetas HTML para personalizar su comportamiento o apariencia.

B

Backend
Parte del desarrollo web que se ejecuta en el servidor.
Browser
Software utilizado para navegar por páginas web.
Box Model
Concepto en CSS que describe cómo se calculan los márgenes, bordes, padding y contenido de un elemento.
Branch
Línea de desarrollo independiente dentro de un repositorio.

C

CSS
Lenguaje de estilo para diseñar páginas web.
Cache
Memoria temporal para mejorar el rendimiento del navegador.
Class
Atributo HTML usado para agrupar elementos con un estilo común.
Commit
Punto de control en el historial de cambios.
Comando
Instrucción que el usuario da al sistema.

D

Display
Propiedad de CSS que define cómo un elemento debe mostrarse en la página.
Div
Etiqueta de contenedor genérica en HTML utilizada para agrupar elementos y aplicarles estilos o propiedades específicas.
Depuración (Debugging)
Proceso de identificar y corregir errores o fallos en el código.
Despliegue (Deployment)
El proceso de publicar un sitio web o una aplicación en un servidor para que sea accesible desde la web.

E

Elementos de Formulario
Elementos de HTML que permiten la interacción con el usuario.
Estructura Semántica (Semantic HTML)
Uso de etiquetas HTML con significado, para mejorar la accesibilidad y la comprensión del contenido.
Espaciado
Concepto en CSS que se refiere al espacio alrededor de los elementos, gestionado con propiedades como padding y margin.
Etiquetas (Tags)
Elementos básicos de HTML que definen la estructura y contenido de un sitio web
Eventos
Acciones que ocurren en la página web, como clics, movimientos del mouse o cambios en un campo de texto.
Especificidad
manera mediante la cual los navegadores deciden qué valores de una propiedad CSS son más relevantes para un elemento y, por lo tanto, serán aplicados.

F

Frontend
arte del desarrollo web visible e interactiva para el usuario, desarrollada con HTML, CSS y JavaScript.
Flexbox
Sistema de diseño en CSS que permite distribuir los elementos de una caja de forma flexible y alinearlos en cualquier dirección (horizontal o vertical).
Font-family
Propiedad CSS utilizada para especificar la fuente tipográfica de un texto.
Formularios
Elementos HTML que permiten la interacción con el usuario para enviar datos a un servidor.
Fork
Copia de un repositorio en la cuenta de otro usuario para modificarlo sin afectar el original

G

Gradiente (Gradient)
Técnica en CSS para crear transiciones suaves entre colores.
Grid
istema de diseño en CSS que permite crear estructuras de filas y columnas para organizar el contenido de manera más compleja que con Flexbo.
Gradiente (Gradient)
Técnica en CSS para crear transiciones suaves entre colores.
Git
Sistema de control de versiones que permite gestionar los cambios en el código fuente de un proyecto de desarrollo web.
GitHub
lataforma en línea que aloja repositorios Git y facilita el trabajo colaborativo en proyectos de desarrollo web.

H

HTML (Hypertext Markup Language)
Lenguaje de marcado utilizado para crear la estructura de las páginas web, mediante el uso de etiquetas y atributos.
Height
Propiedad CSS que define la altura de un elemento.

I

ID
Atributo HTML único que se usa para identificar un solo elemento dentro de la página, lo que permite seleccionarlo y estilizarlo en CSS de manera única..
Input
Elemento HTML utilizado para crear campos de entrada, como texto, contraseñas o botones, en formularios.

L

Link
Etiqueta < a> en HTML utilizada para definir hipervínculos a otras páginas o recursos.
Line-height
ropiedad CSS que define la altura de la línea, afectando el espacio entre las líneas de texto.

M

Margin
Propiedad CSS que define el espacio exterior de un elemento, creando distancia entre él y otros elementos.
Max-width / Min-width
Propiedades CSS que limitan el ancho máximo o mínimo de un elemento, ayudando en el diseño responsivo.
Merge
Proceso para combinar los cambios de dos ramas

N

Node.js
Entorno de ejecución de JavaScript en el servidor basado en el motor V8 de Google Chrome. Permite ejecutar código JavaScript fuera del navegador.

P

Padding
Propiedad CSS que define el espacio interno entre el borde de un elemento y su contenido.
Position
Propiedad CSS que define el tipo de posicionamiento de un elemento, con valores como static, relative, absolute, fixed y sticky.
Pull request
Propuesta para fusionar cambios desde una rama a otra, especialmente útil en trabajo colaborativo.

R

Relative Positioning
Propiedad de posicionamiento CSS que coloca un elemento relativo a su posición normal en el flujo del documento.
Repositorio
Carpeta que contiene el historial de versiones de un proyecto en Git.
Readme
Archivo en formato Markdown que describe un proyecto..

S

Selector
Parte del código CSS que se utiliza para seleccionar el o los elementos HTML a los que se aplicarán los estilos.
Style
Atributo HTML que se utiliza para aplicar estilos en línea a un elemento.

T

Text-align
Propiedad CSS que controla la alineación horizontal del texto dentro de un elemento, con valores como left, center, y right.
Terminal Bash
es un intérprete de comandos utilizado principalmente en sistemas operativos basados en Unix, como Linux y macOS.

U

Underline
Propiedad de texto CSS que agrega un subrayado a un elemento, generalmente texto.
URL
Dirección web que se utiliza para acceder a recursos en la web, como páginas HTML, imágenes, o archivos.
User Interface
nterfaz visual con la que interactúa el usuario en una aplicación web o móvil.

V

Visibility
Propiedad CSS que controla si un elemento es visible o no, pero sin eliminarlo del flujo de la página. El valor hidden lo oculta, pero sigue ocupando su espacio.
Version Control
Sistema que registra los cambios en el código fuente a lo largo del tiempo, permitiendo revertir y gestionar versiones del código.
